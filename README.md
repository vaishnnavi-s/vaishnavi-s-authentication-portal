Authentication System Styling

Project Overview:

This project is a simple Authentication System created using HTML, CSS, and Bootstrap 5. It includes different pages such as login, registration, forgot password, reset password, and dashboard. The main aim of this assignment is to change basic HTML pages into a clean, responsive, and professional-looking application using Bootstrap and custom CSS.

This project shows how front-end technologies can be used to design a simple and organized user interface for authentication pages. It also focuses on improving the design and making the pages responsive so that they work well on devices like desktops, laptops, tablets, and mobile phones.

Technologies Used:

The following technologies were used in this project:
1. HTML – Used to create the structure of the web pages.
2. CSS – Used to add styling and improve the look of the pages.
3. Bootstrap 5 – Used to create responsive layouts and components like cards, forms, buttons, and navigation bars.
4. Google Fonts – Used to improve the text style and make the design look better.

The authentication system has five main pages:

Login Page
![Login Page](screenshots/login.png)
The login page allows users to enter their email and password to log into the system. It uses a Bootstrap card and styled form fields to create a clean login interface. It also provides a link for users who forgot their password.

Registration Page
![Register Page](screenshots/register.png)
The registration page allows new users to create an account by entering their details such as name, email, password, and confirm password. Bootstrap form classes are used to arrange the fields neatly.

Forgot Password 
![Forgot Password](screenshots/forgot-password.png)
This page allows users to enter their email if they forget their password. It provides a simple form where users can request a password reset.

Reset Password Page
![Reset Password](screenshots/reset-password.png)
The reset password page allows users to set a new password after requesting a password reset. It includes fields for entering a new password and confirming it.

Dashboard Page
![Dashboard](screenshots/dashboard.png)
The dashboard page appears after a successful login. It displays a welcome message and a navigation bar. This page acts as the main page after the user logs in.

Features:
This project includes several features to make the interface simple and user-friendly:
1. Responsive design using Bootstrap
2. Clean login and registration forms
3. Easy navigation between pages
4. Custom CSS styling for better appearance
5. Card layout for a neat design
6. Background styling and proper spacing
7. Mobile-friendly design for different screen sizes
8. Simple and easy-to-use authentication interface
