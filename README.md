Vaishnavi S – HTML Authentication System

1. Project Title:
- HTML Multi-Page Authentication Prototype

2. Project Description:
- This project is a simple authentication system developed using pure HTML.  
- It demonstrates structured navigation between multiple pages without using CSS or JavaScript.

3. Pages Included:
- login.html  
- register.html  
- forgot-password.html  
- reset-password.html  
- dashboard.html  
- README.md  

4. Navigation Flow:
- Login → Dashboard  
- Login → Register  
- Login → Forgot Password  
- Forgot Password → Reset Password  
- Reset Password → Login  
- Register → Login  
- Dashboard → Logout → Login  

5. HTML Concepts Used:
- Semantic elements (`header`, `main`, `section`, `footer`)  
- Form elements (`form`, `input`, `label`)  
- Anchor tag (`<a>`) for redirection  
- Proper `for` and `id` attributes for accessibility  

6. Tools Used:
- HTML5  
- Visual Studio Code  
- Git (Version Control)  
- GitHub (Repository Hosting)  
- Web Browser for testing  

7. Objective of Assignment:
- Understand multi-page website structure  
- Learn page-to-page navigation  
- Practice form creation  
- Use semantic HTML  
- Manage project using Git and GitHub  

8. Restrictions Followed:
- No CSS used  
- No JavaScript used  
- Only HTML structure and anchor navigation implemented  

9. Learning Outcomes:
- Improved understanding of HTML structure  
- Gained experience in multi-page linking  
- Learned proper Git commit practices  
- Developed structured project organization
